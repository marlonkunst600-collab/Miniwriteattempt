Understanding the __experimentalToolsPanel component

If you have looked at the "Dimensions" or "Border" panels in Core blocks, you have probably noticed a very distinct behavior.

They are incredibly clean by default, but you can click a "plus" or "vertical ellipsis" icon to reveal specific controls (like margin, padding, and radius). They also feature a "Reset" button that appears only when a value has been changed.

Most third-party developers simply dump a dozen RangeControl components into a PanelBody. This creates a cluttered, overwhelming experience for beginners.

Using ToolsPanel allows you to implement "Progressive Disclosure," making your block feel like a native WordPress tool.

The Snippet: A Professional "Overlay" Panel

This code creates a sidebar panel where the "Opacity" slider is hidden behind a menu by default. This keeps the UI tidy for basic users while remaining fully accessible for power users.

File: edit.js 
import { __ } from '@wordpress/i18n';
import { useBlockProps, InspectorControls, PanelColorSettings } from '@wordpress/block-editor';
import { 
    __experimentalToolsPanel as ToolsPanel, 
    __experimentalToolsPanelItem as ToolsPanelItem,
    RangeControl 
} from '@wordpress/components';

export default function Edit( { attributes, setAttributes } ) {
    const { overlayColor, opacity } = attributes;

    return (
        <>
            <InspectorControls>
                {/* The ToolsPanel replaces a standard PanelBody */}
                <ToolsPanel label={ __( 'Overlay Settings' ) }>
                    
                    {/* Item 1: Always visible */}
                    <ToolsPanelItem
                        label={ __( 'Color' ) }
                        isShownByDefault={true}
                        hasValue={ () => !! overlayColor }
                        onReset={ () => setAttributes({ overlayColor: undefined }) }
                    >
                        <PanelColorSettings
                            title={ __( 'Overlay Color' ) }
                            colorSettings={ [ {
                                value: overlayColor,
                                onChange: (color) => setAttributes({ overlayColor: color }),
                                label: __( 'Color' ),
                            } ] }
                        />
                    </ToolsPanelItem>

                    {/* Item 2: Progressive Disclosure (Hidden by default) */}
                    <ToolsPanelItem
                        label={ __( 'Opacity' ) }
                        isShownByDefault={false} // The secret to a clean UI
                        hasValue={ () => opacity !== 0.5 }
                        onReset={ () => setAttributes({ opacity: 0.5 }) }
                    >
                        <RangeControl
                            label={ __( 'Opacity' ) }
                            value={ opacity }
                            onChange={ (val) => setAttributes({ opacity: val }) }
                            min={0} max={1} step={0.1}
                        />
                    </ToolsPanelItem>

                </ToolsPanel>
            </InspectorControls>

            <div { ...useBlockProps({
                style: { backgroundColor: overlayColor, opacity: opacity }
            }) }>
                { __( 'Overlay Content' ) }
            </div>
        </>
    );
}
