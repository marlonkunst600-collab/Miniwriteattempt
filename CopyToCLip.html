Adding a "Copy to Clipboard" Action to the Block Toolbar

Many beginners think the toolbar is only there to align text or handle bold/italic styling, but you can add a significant amount of custom functionality to it.

"It's especially useful for functions that your users should be able to access quickly without clicking through countless menus."

This snippet adds a custom button to the floating toolbar of your block that allows users to instantly copy the block's text content. This is highly effective for blocks that display "Code Snippets", "Referral Codes", or "API Keys".

JavaScript Component: edit.js 
import { useBlockProps, BlockControls } from '@wordpress/block-editor';
import { ToolbarGroup, ToolbarButton } from '@wordpress/components';
import { copy } from '@wordpress/icons'; // Using a core WordPress icon

export default function Edit({ attributes }) {
    const { content } = attributes;

    const copyToClipboard = () => {
        navigator.clipboard.writeText(content);
        alert('Copied to clipboard!');
    };

    return (
        <>
            {/* This puts our button in the floating toolbar */}
            <BlockControls group="other">
                <ToolbarGroup>
                    <ToolbarButton
                        icon={copy}
                        label="Copy Content"
                        onClick={copyToClipboard}
                    />
                </ToolbarGroup>
            </BlockControls>

            <div {...useBlockProps()}>
                <p>Content to copy: <strong>{content}</strong></p>
            </div>
        </>
    );
}
