Precision Layouts with BoxControl

If you have ever tried to build a custom block with adjustable padding or margins, you probably know that it isn't exactly easy.

Every time I have tried it without native components, the results were cluttered, confusing, and didn't match the Core WordPress UI. That’s why BoxControl has been a real savior.

What is BoxControl?

BoxControl is a high-level component from the @wordpress/components package designed specifically for managing dimensions: Top, Right, Bottom, and Left.

It provides a polished visual interface that allows users to link all sides for uniform spacing or unlink them to set individual values for each side.

No longer Experimental

For the last few years, this component used the __experimentalBoxControl prefix. As of WordPress 6.7, the component has officially stabilized as BoxControl. You should transition your modern projects to the stable import.

Quick Implementation

To use BoxControl, you must store your values as an object in your block attributes.

File: edit.js 
import { BoxControl } from '@wordpress/components';

export default function Edit({ attributes, setAttributes }) {
    const { spacing } = attributes;

    return (
        <BoxControl
            label="Custom Padding"
            values={ spacing }
            onChange={ (nextValues) => setAttributes({ spacing: nextValues }) }
            allowReset={ true }
            units={ [ 'px', 'em', 'rem', 'vw' ] }
        />
    );
}
